  • Triángulo equilátero invertido (todos los lados iguales)
  • Distancia entre vértices: d = 320px
  • Radio mayor = d = 320px (igual a distancia entre vértices)
  • Radio menor = d / 1.5 = 213.33px (CORRECCIÓN APLICADA)
  • Radio medio = (radio mayor + radio menor) / 2 = 266.67px

  • Círculos solo con bordes, sin relleno de color
